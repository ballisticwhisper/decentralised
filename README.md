### Build on Linux

##### Install Git

```
sudo apt-get install git
```

##### Clone repository and submodules

```
git clone https://github.com/ballisticwhisper/decentralised.git dc
git submodule init
git submodule update
```

##### Install CMake

```
sudo apt-get install cmake
```
